## **Local LLM for OpenMetadata (**VSCODE EXTENSION)


- [https://marketplace.visualstudio.com/items?itemName=MarkusBegerow.local-llm-chat-vscode-openmetadata](https://marketplace.visualstudio.com/items?itemName=MarkusBegerow.local-llm-chat-vscode-openmetadata)
- Repo https://github.com/markusbegerow/local-llm-chat-vscode-openmetadata/tree/main

	Configurazione nei Settings VsCode
	```
	{
	  "openmetadataExplorer.llm.provider": "custom",
	  "openmetadataExplorer.llm.custom.endpoint": "http://localhost:1234/v1",
	  "openmetadataExplorer.llm.custom.apiKey": "optional-api-key",
	  "openmetadataExplorer.llm.custom.model": "model-name",
	  "openmetadataExplorer.openmetadataUrl": "http://localhost:8585",
	  "openmetadataExplorer.openmetadataAuthToken": "YOUR_BOT_TOKEN"
	}
	```
